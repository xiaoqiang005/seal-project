# Seal Project

这是一个基于 Vue.js 和 Django 的项目。

## 项目文档

您可以在以下地址查看项目文档：
https://xiaoqiang005.github.io/seal-project/

## 技术栈

- 前端：Vue 3 + TypeScript
- 后端：Django + Django REST framework
- 数据库：MySQL

## 开发指南

### 前端开发

```bash
cd frontend
npm install
npm run dev
```

### 后端开发

```bash
cd backend1
pip install -r requirements/dev.txt
python manage.py runserver
```
